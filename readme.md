# Animated Easing Cheat Sheet

30種類の標準的なイージング関数 (In, Out, InOut) をアニメーションで視覚化し、クリックすることで各イージング関数のコードをクリップボードにコピーできるチートシートである。

## 特徴

- p5.js を用いたスムーズなアニメーション表示
- 各グラフへのホバーによる数式のプレビュー表示
- 左クリックによる関数コードのコピー機能
- Vite と TypeScript を用いたモダンな開発環境

## ファイル構造

本プロジェクトのディレクトリおよびファイル構造は以下の通りである。

```
.
├── index.html
├── package.json
├── tsconfig.json
├── vite.config.ts
├── .gitignore
├── readme.md
└── src
    ├── main.ts         - アプリケーションのエントリーポイント。CSS とスケッチファイルをインポートする。
    ├── style.css       - アプリケーションのスタイルシート。レイアウトとツールチップなどの装飾を担当する。
    ├── sketch.ts       - p5.js のインスタンスモードを用いた描画およびイベント処理の統括を行うスクリプト。
    ├── easingCell.ts   - 各イージンググラフの描画、アニメーション状態の更新、およびコードのクリップボードへのコピー処理を行うクラス。
    └── easings.ts      - 標準イージング関数の定義、およびコピー用の数式テキストをまとめたデータモジュール。
```

## セットアップと起動

### 1. 依存関係のインストール

pnpm を使用して依存関係をインストールする。

```bash
pnpm install
```

### 2. 開発サーバーの起動

ローカルサーバーを起動する。

```bash
pnpm dev
```

### 3. プロダクションビルド

ビルドを実行し、配布用ファイルを `dist` ディレクトリに出力する。

```bash
pnpm build
```
